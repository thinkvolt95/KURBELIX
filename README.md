# DIY Cycling Power Meter

A fork of Svenbosma/DIY-Garmin-Powermeter.

---

# Changes from oroginal

- Cadence calculation using a sensor fusion of the gyro and the accelerometer
- Using the accelerometer to measure the actual crank angle; so 0° degree is always top dead-spot
- Added Low Pass Filtering for gyro and accelerometer
- Added Cycling Power Vector from CPP profile, currently not tested
